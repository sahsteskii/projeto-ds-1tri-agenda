 ## descrição do sistema
 esse projeto é um aplicativo de agendamentos, ele tem como objetivo facilitar o agendamentos de compromissos de forma simples e rápida.

 o aplicativo permite que o usuário faça login com o google, acessem seu perfil e escolham datas e horários disponíveis de acordo com o interesse, e também tem a central de ajuda para que eles possam enviar um feedback quando precisarem.

 ## funcionalidades principais 

 sistema de login de usuários;
 visualização e edição de perfil;
 seleção de datas disponíveis;
 seleção de horas disponíveis;
 realização de agendamentos;
 central de ajuda ao usuário;

 ## tecnologias utilizadas
 node.js
 react native
 mysql
